## Social Network Ads Purchase Prediction (Logistic Regression)
This project uses Logistic Regression (scikit-learn) to predict whether a user will purchase a product based on Age, Estimated Salary, and Gender from the Social Network Ads dataset.

It also demonstrates:
Data preprocessing
Feature scaling
Train/test split
Model evaluation
Hyperparameter tuning using GridSearchCV
Data visualization

## Dataset
File: `Social_Network_Ads.csv`

Features used:
Age
EstimatedSalary
Gender

## Target:
Purchased (0 = No, 1 = Yes)

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
